# 图书管理系统（C++）

一个基于控制台的图书借阅管理系统，支持图书的增删查、借阅/归还状态管理。

## ✨ 功能

- 添加图书（自动去重，ISBN 唯一）
- 按 ISBN 查询图书信息（书名、作者、状态）
- 借书（检查是否可借）
- 还书（检查是否已借出）
- 打印所有图书列表
- 交互式菜单循环操作

## 🛠 构建与运行

### 依赖
- C++17 兼容编译器（如 g++, clang++）
- CMake ≥ 3.10

### 步骤
```bash
# 克隆或进入项目目录
cd book_manager

# 创建构建目录
mkdir build && cd build

# 生成构建文件
cmake ..

# 编译
make

# 运行
./book_manager
